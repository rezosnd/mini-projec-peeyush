KIITease

Next-Gen Project Management System · Technical Report

Executive Summary

KIITease is a comprehensive collaboration portal designed for the Kalinga Institute of Industrial Technology (KIIT). It bridges the gap between academic project groups and faculty mentors through a type-safe, server-rendered platform. The system focuses on transparency, real-time synchronization, and a "Managed Portal" logic for role-based oversight.

2. Key Methodology

2.1 Group Syndicate Model

Students can form persistent groups with their peers. This mapping allows the system to treat a "Group" as a single entity for task assignments, ensuring all members share progress and submission history.

2.2 Task Lifecycle

  • Assignment: Faculty target tasks to individual Roll Numbers or predefined Groups.
  • Tracking: Students update a 0-100% progress slider and manage granular to-do items.
  • Submission: Triple-link verification (Report, PPT, and GitHub Repository).

2.3 Managed Portal Logic

The system utilizes a dual-path routing engine. The /student portal emphasizes individual and group task completion, while the /teacher portal provides a macro view of assignment status and progress across all sections.
