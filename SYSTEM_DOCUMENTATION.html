KIIT Mini Project Management System

Official Documentation & User Guide

1. Executive Summary

The KIIT Mini Project Management System is a premium, "Managed Portal" designed to streamline the project lifecycle for students and faculty. It combines high-end aesthetics with robust collaboration tools, facilitating seamless task management, group coordination, and progress tracking.

2. Technology Stack

The system leverages a modern, serverless, and type-safe architecture.

Layer Technology Purpose
Framework Next.js 15 (App Router) SSR, Routing, Server Actions
Database Neon PostgreSQL Relational Data & Integrity
Security Jose (JWT) + Bcrypt Auth & Data Protection
Design Vanilla CSS + Syne Premium Managed UI

3. Key Features

Collaboration Nodes

Students can form persistent groups, allowing for unified project submissions and collective progress tracking. Teachers can target assignments to either individuals or entire teams.

Progress Synchronization

A real-time synchronization layer allows students to report completion percentages (0-100%) and manage milestones via integrated to-do lists, providing faculty with instant oversight.

4. User Experience

The interface follows a "Managed Portal" logic, utilizing a two-column distribution that separates navigational controls from core project streams. This ensures clarity even when managing multiple complex tasks.
